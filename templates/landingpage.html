GABAY Chatbot

The GABAY Chatbot is a smart, real-time virtual assistant that provides guidance and emergency support during crises like typhoons, floods, or fires. It uses advanced Natural Language Processing (NLP) to understand and respond to user needs effectively

Key Features:

24/7 real-time emergency guidance

Context-aware replies based on disaster type

Quick access to emergency hotlines

Integrated alerts and disaster warnings

Nearest Emergency Area (NEA)

The Nearest Emergency Area (NEA) feature uses real-time GPS to help users locate the closest safe zones and emergency services.

Key Features:

Live location tracking with GPS

Shows evacuation centers, hospitals, police and fire stations

Interactive map with route suggestions

Dynamic updates based on changing hazards

Dashboard

This dynamic dashboard provides visual insights into typhoon, flooding, and fire-related incidents in Pasig City. It aids both public users and government agencies for better emergency response.

Key Features:

Visual reports: charts, graphs, and heatmaps

Real-time updates from verified sources

Filter by incident type, date, or barangay

Community-sourced reporting (with verification)
